Online examination file submission system
Due to the pandemic, the University of South Africa (UNISA) has decided to conduct all examinations in an online environment.

The purpose of this project is then to look at the digital environment and design and develop a simulation of such a system, using the real-life scenario and environment.

You thus must design and develop an online examination file submission system, that can be presented to the University to be used as an alternative to the current system.
